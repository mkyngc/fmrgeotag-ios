FMR Geotag

Version 1.0.0

FMR Geotag is an offline-first field data collection tool for geotagging Farm-to-Market Roads. It supports fixed-interval 50-meter segment surveys as well as continuous pathway (track) recording, complete with photo capture, GPS metadata, and multi-format export.

All data is stored locally on the device using SQLite and the device file system. No internet connection is required at any point during data collection.

Credits

Built with Apache Cordova, Bootstrap 5, Leaflet, and JSZip.
